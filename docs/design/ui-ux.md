## 1. 디자인 철학

### 핵심 키워드

- **따뜻한 (Warm)**: 부드러운 색감, 둥근 모서리, 포근한 느낌
- **귀여운 (Cute)**: 아기자기한 아이콘, 이모티콘, 캐릭터
- **직관적 (Intuitive)**: 토스처럼 미니멀하면서 명확한 UI
- **감성적 (Emotional)**: 독서의 즐거움을 시각적으로 표현

### 마스코트 캐릭터: 밍들레씨

- **모티프**: 민들레 홀씨 — 하얀 솜뭉치 형태, 둥글고 폭신한 2등신 캐릭터
- **철학**: "작고 둥근 생각들, 바람따라 날아다니며 모으는 하루들"
- **역할**: 앱 전반의 마스코트 + 독서 에이전트 "밍들레"의 비주얼 아이덴티티
- **상세 가이드**: [character.md](character.md) 참조

### 참고 스타일

| 서비스              | 참고 요소                                                      |
| ------------------- | -------------------------------------------------------------- |
| **밍들레씨 인스타** | 캐릭터 디자인, 톤앤매너, 감성적 일러스트, 굵은 아웃라인 스타일 |
| **카카오**          | 카드형 레이아웃, 따뜻한 색감, 둥근 모서리, 캐릭터 활용         |
| **토스**            | 미니멀, 직관적 네비게이션, 마이크로 인터랙션, 타이포그래피     |
| **Repov**           | 다양한 뷰 모드 (갤러리/캘린더/타임라인), 미니 블로그형 카드    |

## 4. 컴포넌트 스타일

### 카드 (핵심 UI 요소)

```
- 둥근 모서리: rounded-2xl (16px)
- 부드러운 그림자: shadow-sm
- 패딩: p-4 ~ p-6
- 배경: white
- 호버: 약한 스케일 업 (scale-[1.02])
```

### 버튼

```
- Primary: 그린 배경, 흰 텍스트, rounded-xl (밍들레씨 두건 색)
- Accent: 노란 배경, 짙은 텍스트, rounded-xl (민들레꽃 CTA)
- Secondary: 그린 테두리, 그린 텍스트, rounded-xl
- Ghost: 배경 없음, 호버 시 연한 그린 배경
- 아이콘 버튼: 원형, 48px
```

### 입력 필드

```
- rounded-xl (12px)
- 포커스: 그린 테두리 + 연한 그린 배경
- 플레이스홀더: 따뜻한 회색
```

## 6. 핵심 뷰/페이지

### 6.1 홈 (로그인 전)

- 후드 쓴 밍들레씨 일러스트 + 서비스 소개
- "작고 둥근 독서 모임, 밍들" 카피 + 로그인 CTA

### 6.2 홈 (로그인 후)

- 내 모임 목록 (카드형)
- 다가오는 모임 일정
- 최근 활동 피드
- AI 에이전트 플로팅 버튼

### 6.3 모임 상세

- **갤러리 뷰**: 책 표지 그리드 (날짜별 구분)
- **캘린더 뷰**: 월간 캘린더 + 모임 일정 표시
- **타임라인 뷰**: 시간순 모임 활동 (Phase 2)
- 뷰 전환 탭 (상단)

### 6.4 세션 상세

- 책 정보 카드 (표지 + 제목 + 저자)
- 발제문 (마크다운 지원)
- 사진 갤러리 (Phase 2)
- 유투브 임베딩 (Phase 2)
- 후기 목록 + 작성

### 6.5 AI 에이전트 "밍들레" 채팅

- **위치**: 하단 우측 플로팅 버튼 (밍들레씨 아바타)
- **UI**: 슬라이드업 패널 (카카오톡 느낌)
- **아바타**: 밍들레씨 기본 미소 표정 (48x48)
- **구성**: 밍들레 프로필 헤더 + 메시지 목록 + 입력창 + 전송 버튼
- **특징**: 스트리밍 타이핑 효과, 밍들레씨 톤의 이모티콘 활용
- **빈 상태**: 밍들레씨 책 읽는 포즈 + "무엇이든 물어보세요!"
- **상세 캐릭터 적용**: [character.md](character.md) 6장 참조

## 7. 반응형 디자인

### 브레이크포인트

```
Mobile:  < 640px   (기본, 모바일 퍼스트)
Tablet:  640~1024px
Desktop: > 1024px
```

### 모바일 우선 설계

- 독서 모임은 모바일 사용이 주력
- 모든 뷰를 모바일에서 먼저 설계
- 데스크톱은 여백과 멀티 컬럼 추가

## 8. 마이크로 인터랙션

| 이벤트         | 밍들레씨 반응      | 메시지                            |
| -------------- | ------------------ | --------------------------------- |
| 독서 완료      | 민들레꽃 들고 축하 | "노란 꽃이 피어나고 있어요! 🌼"   |
| 후기 작성      | 하트 둥둥 포즈     | "대단해요! 기록은 추억이 되죠 ❤️" |
| 발제문 작성    | 책상 앞 박수       | "멋진 발제문이에요! ✨"           |
| 연속 모임 참석 | 사자 거울 오마주   | "나는 독서왕이다! 🦁"             |
| 첫 모임 생성   | 후드 쓴 밍들레씨   | "모험이 시작됐어요! 🌱"           |
| 로딩           | 민들레 홀씨 날리기 | "잠깐만요, 찾아보는 중..."        |

```
밍들레씨 인스타 톤 — 짧고 귀여우며 공감 가는 1인칭 말투:
- "오늘도 한 페이지 읽었어요! 아이캔두잇 💪"
- "발제문 작성 완료! 나는 독서왕이다! 🦁"
- "모임이 다가오고 있어요. 설레지 않나요? 🌼"
- "이번 달 벌써 2권째! 노란 꽃이 곧 피어날 거예요 ✨"
- "아직 후기를 안 쓰셨네요... 한 줄이라도 좋아요!"
- "잘 버티고 잘 읽어내자 📖"
```
