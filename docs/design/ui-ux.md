## 1. 디자인 철학

### 핵심 키워드
- **따뜻한 (Warm)**: 부드러운 색감, 둥근 모서리, 포근한 느낌
- **귀여운 (Cute)**: 아기자기한 아이콘, 이모티콘, 캐릭터
- **직관적 (Intuitive)**: 토스처럼 미니멀하면서 명확한 UI
- **감성적 (Emotional)**: 독서의 즐거움을 시각적으로 표현

### 참고 스타일
| 서비스 | 참고 요소 |
|--------|---------|
| **카카오** | 카드형 레이아웃, 따뜻한 색감, 둥근 모서리, 캐릭터 활용 |
| **토스** | 미니멀, 직관적 네비게이션, 마이크로 인터랙션, 타이포그래피 |
| **Repov** | 다양한 뷰 모드 (갤러리/캘린더/타임라인), 미니 블로그형 카드 |

## 4. 컴포넌트 스타일

### 카드 (핵심 UI 요소)
```
- 둥근 모서리: rounded-2xl (16px)
- 부드러운 그림자: shadow-sm
- 패딩: p-4 ~ p-6
- 배경: white
- 호버: 약한 스케일 업 (scale-[1.02])
```

### 버튼
```
- Primary: 보라 배경, 흰 텍스트, rounded-xl
- Secondary: 보라 테두리, 보라 텍스트, rounded-xl
- Ghost: 배경 없음, 호버 시 연한 보라 배경
- 아이콘 버튼: 원형, 48px
```

### 입력 필드
```
- rounded-xl (12px)
- 포커스: 보라 테두리 + 연한 보라 배경
- 플레이스홀더: 따뜻한 회색
```

## 6. 핵심 뷰/페이지

### 6.1 홈 (로그인 전)
- 서비스 소개 + 로그인 CTA
- 따뜻한 일러스트/이모티콘

### 6.2 홈 (로그인 후)
- 내 모임 목록 (카드형)
- 다가오는 모임 일정
- 최근 활동 피드
- AI 에이전트 플로팅 버튼

### 6.3 모임 상세
- **갤러리 뷰**: 책 표지 그리드 (날짜별 구분)
- **캘린더 뷰**: 월간 캘린더 + 모임 일정 표시
- **타임라인 뷰**: 시간순 모임 활동 (Phase 2)
- 뷰 전환 탭 (상단)

### 6.4 세션 상세
- 책 정보 카드 (표지 + 제목 + 저자)
- 발제문 (마크다운 지원)
- 사진 갤러리 (Phase 2)
- 유투브 임베딩 (Phase 2)
- 후기 목록 + 작성

### 6.5 AI 에이전트 채팅
- **위치**: 하단 우측 플로팅 버튼
- **UI**: 슬라이드업 패널 (카카오톡 느낌)
- **구성**: 메시지 목록 + 입력창 + 전송 버튼
- **특징**: 스트리밍 타이핑 효과, 이모티콘 활용

## 7. 반응형 디자인

### 브레이크포인트
```
Mobile:  < 640px   (기본, 모바일 퍼스트)
Tablet:  640~1024px
Desktop: > 1024px
```

### 모바일 우선 설계
- 독서 모임은 모바일 사용이 주력
- 모든 뷰를 모바일에서 먼저 설계
- 데스크톱은 여백과 멀티 컬럼 추가

## 8. 마이크로 인터랙션

### 격려/축하 인터랙션
| 이벤트 | 인터랙션 |
|--------|---------|
| 독서 완료 | 축하 애니메이션 + 따뜻한 메시지 |
| 후기 작성 | "대단해요! 기록은 추억이 되죠" |
| 발제문 작성 | "멋진 발제문이에요!" |
| 연속 모임 참석 | "꾸준함이 최고예요!" + 뱃지 (Phase 3) |

### UX 카피 톤
```
따뜻하고 격려하는 톤:
- "오늘도 한 페이지 읽었어요!"
- "발제문 작성 완료! 대단해요"
- "모임이 다가오고 있어요. 설레지 않나요?"
- "이번 달 벌써 2권째! 대단해요"
```
