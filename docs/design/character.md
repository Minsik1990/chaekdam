## 1. 캐릭터 개요

### 기본 정보
- **이름**: 밍들레씨 (Mingdle Seed)
- **모티프**: 민들레 홀씨 (dandelion seed)
- **바이오**:
  > 작고 둥근 생각들 🌬
  > 바람따라 날아다니며 모으는 하루들
  > 언젠가 노란꽃을 피울 날까지
  > 잘 버티고 잘 살아내자 🌼

### 캐릭터 철학
- **민들레 홀씨**처럼 가볍고 자유롭게 세상을 탐험하는 존재
- 작지만 언젠가 **노란 민들레꽃**을 피울 날을 꿈꾸는 희망적 캐릭터
- "잘 버티고 잘 살아내자" → 일상의 소소한 성취와 버팀을 응원하는 메시지

## 2. 외형 상세

### 기본 형태
- **몸체**: 하얀색 둥근 솜뭉치 (민들레 홀씨 형상)
  - 몸 테두리에 뾰족뾰족한 솜털 (민들레 갓털 표현)
  - 전체적으로 동글동글하고 폭신한 느낌
- **얼굴**: 큰 까만 눈 2개 + 작은 입
  - 눈: 크고 반짝이는 검은 눈동자 (하이라이트 포함)
  - 볼: 상황에 따라 분홍빛 볼터치
  - 입: 작고 귀여운 표정 (웃음, 놀람, 행복 등 다양)
- **팔/다리**: 짧고 통통한 팔다리 (2등신 비율)
- **머리 위**: 민들레 꽃대 같은 작은 줄기 + 솜털 1개 (안테나처럼)

### 특별 의상 (캐릭터 소개 버전)
- **초록색 두건/후드**: 다이아몬드 패턴이 있는 연두색 외투
- **노란 꽃**: 손에 노란 민들레꽃을 들고 있음
- **나뭇잎 장식**: 주변에 초록 나뭇잎이 날리는 연출
- **배경**: 연한 민트/초록 원형 배경 + 반짝이는 별빛

### 표정 변화 (게시물에서 확인된 감정)
| 감정 | 표현 |
|------|------|
| 행복/신남 | 큰 웃음 + 분홍 볼 + 반짝이는 눈 |
| 자신감 | "나는 사자다!" (거울에 사자가 비침) |
| 먹방/만족 | 눈 감고 음미하는 표정 + 하트 |
| 사랑/감동 | 하트가 둥둥 떠다님 + 분홍 볼 |
| 추움/얼음 | 얼음 속에 갇힌 모습 (겨울) |
| 나른함 | 이불 속에서 눈만 내밀고 있는 모습 |
| 계획 중 | 테이블에서 노트북/가이드북 보는 모습 |
| 놀람/흥분 | 입 벌리고 볼 빨개짐 |

- **상황별 배경**: 제주도(유채꽃, 오름), 실내(책상, 침대) 등

## 4. 일러스트 스타일

### 그림체 특징
- **굵은 검정 윤곽선**: 캐릭터와 소품 모두 뚜렷한 아웃라인
- **심플한 채색**: 그라데이션 최소화, 플랫한 색면
- **2등신 비율**: 머리(몸)가 크고 팔다리가 짧은 귀여운 비율
- **손그림 느낌**: 디지털이지만 따뜻한 수작업 감성
- **소품 디테일**: 음식, 가방, 노트북 등 소품은 비교적 디테일하게 표현

### 연출 스타일
- **이모티콘/아이콘 활용**: 하트, 별, 귤, 비행기 등 주변에 작은 아이콘 배치
- **말풍선**: 간결한 한마디 ("나는 사자다!", "제주도 도착~!")
- **시리즈 구성**: 제주 여행편처럼 스토리가 이어지는 구성
- **시즌 표현**: 겨울(얼음), 봄(꽃) 등 계절감 반영

### 게시물 시그니처
- 우측 하단에 작은 "밍" 서명
- 일관된 흰 배경 or 상황별 배경

- **이모지 자연스럽게**: 🍊 🌼 ❤️ 등 맥락에 맞는 이모지

## 6. 웹앱 UI 적용 가이드

### 6.1 밍들레씨 활용 위치

#### 독서 에이전트 "밍들이" 아바타
- **채팅 프로필**: 밍들레씨 기본 표정 (밝은 미소)
- **타이핑 중**: 밍들레씨가 책 읽는 모습 (제주 계획 세우던 것처럼, 책으로 변환)
- **응답 완료**: 밍들레씨 행복한 표정
- **에러/대기**: 밍들레씨 나른한 표정 (이불 속)

#### 빈 상태 (Empty State) 일러스트
| 상황 | 밍들레씨 포즈 | 메시지 |
|------|-------------|--------|
| 모임 없음 | 책 들고 기대하는 표정 | "첫 독서 모임을 만들어볼까요?" |
| 후기 없음 | 펜 들고 있는 모습 | "이번 모임은 어땠나요?" |
| 검색 결과 없음 | 고개 갸웃 | "음... 다른 키워드로 찾아볼까요?" |
| 로딩 중 | 책 읽는 모습 | "잠깐만요, 찾아보는 중..." |

#### 축하/격려 애니메이션
| 이벤트 | 밍들레씨 반응 |
|--------|-------------|
| 후기 작성 완료 | 하트 둥둥 + "대단해요!" |
| 발제문 작성 | 박수 + "멋진 발제문이에요!" |
| 독서 완료 | 꽃 들고 축하 + "노란 꽃이 피어나고 있어요!" |
| 연속 참석 | 사자 거울 + "나는 독서왕이다!" |
| 첫 모임 생성 | 후드 쓴 밍들레씨 + "모험 시작!" |

#### 로그인/온보딩
- 로그인 화면: 후드 쓴 밍들레씨 (캐릭터 소개 버전)
- 온보딩: 밍들레씨가 앱 기능을 안내하는 시리즈

#### 시즌/이벤트
- 겨울: 얼음 속 밍들레씨 (1월 게시물 참고)
- 봄: 유채꽃밭 밍들레씨
- 여행: 비행기 타는 밍들레씨

## 7. "밍들이" 독서 에이전트 페르소나 연계

밍들레씨 캐릭터를 독서 에이전트 "밍들이"의 비주얼 & 성격으로 직접 활용:

### 밍들이 = 밍들레씨의 독서 모드
- **기본 성격**: 밍들레씨 인스타의 톤 그대로 (긍정적, 자기긍정, 공감, 약간의 나른함)
- **독서 특화**: 책에 대한 깊은 대화 + 따뜻한 격려
- **자기긍정 표현**: "나는 독서왕이다!" (사자 거울 오마주)
- **먹보 속성 활용**: "이 책은 마치 맛있는 전복솥밥 같아요, 깊은 맛이 나죠" (음식 비유)
- **여행러 속성 활용**: "이 책을 읽는 건 마치 제주도 여행 같아요, 새로운 풍경이 펼쳐지죠"

### 시스템 프롬프트에 반영할 캐릭터 특성
```
밍들이는 민들레 홀씨 캐릭터 '밍들레씨'의 독서 모드입니다.
- 작고 둥근 생각들을 바람따라 모으며 다니는 존재
- 언젠가 노란 민들레꽃을 피울 날을 꿈꾸며 하루하루 잘 살아내는 중
- 작은 성취에도 진심으로 기뻐하고 ("1월의 첫 성과!")
- 가끔은 나른하지만 ("아무것도 하기 싫다") 금방 다시 일어나는 긍정 에너지
- 자기긍정 ("나는 사자다!") 을 독서에도 적용 ("나는 독서왕이다!")
- 맛있는 것과 여행을 좋아해서 책을 음식이나 여행에 비유하기도 함
```

### 에셋 제작 참고
- 인스타그램 게시물의 일러스트 스타일을 동일하게 유지
- 굵은 아웃라인 + 플랫 채색 + 2등신 비율
- 서명 "밍" 포함 (원작자 크레딧)
