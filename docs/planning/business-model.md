# 비즈니스 모델 & 성장 전략

> 독독의 비즈니스 비전, 수익화 모델, 성장 전략, KPI

---

## 1. 서비스 비전

### 미션

"모든 독서인이 읽은 책의 감동을 쉽게 기록하고, 함께 나눌 수 있는 공간을 만든다."

### 장기 목표

| 기간  | 목표               | MAU     | 핵심 지표                |
| ----- | ------------------ | ------- | ------------------------ |
| 6개월 | MVP 검증           | 50명    | 주간 기록 작성률 30%+    |
| 1년   | 커뮤니티 형성      | 500명   | 모임 활성도, 리텐션 40%+ |
| 3년   | 한국 독서 기록 1위 | 5,000명 | 기록 누적 10만건+        |

---

## 2. 타겟 사용자

### 핵심 페르소나

**1차 타겟: 기록형 독서인 (Primary)**

- 20~30대, 월 1~3권 독서
- 읽은 책의 감상을 어딘가에 남기고 싶은 욕구
- 노션/메모앱에 기록하지만 체계적이지 않음
- 핵심 니즈: 간편한 기록, 예쁜 UI, 독서 통계

**2차 타겟: 독서 모임 참여자 (Secondary)**

- 25~40대, 월 1~2권 (모임 기반)
- 오프라인 독서 모임 참여 중
- 모임 기록/후기 관리에 어려움
- 핵심 니즈: 모임 관리, 토론 주제 생성, 세션 기록

### 비타겟 (Not Target)

- 전문 서평가/북튜버 (높은 커스터마이징 니즈)
- 다독가/속독가 (기록보다 권수 추적 중심)
- 학술 독서 (논문/학술서 분석)

## 3. 가치 제안 (Value Proposition)

### 핵심 차별점

| 차별점           | 독독                              | 경쟁 서비스                |
| ---------------- | --------------------------------- | -------------------------- |
| AI 대화형 기록   | AI가 질문하며 생각을 끌어냄       | 빈 화면에 직접 작성        |
| AI 독서 지원     | 토론 주제, 발제문, 분석 자동 생성 | 수동 준비                  |
| 무료 + 모임 관리 | 완전 무료 기록 + 모임 통합        | 트레바리(유료), 밀리(구독) |
| 자연적 디자인    | 파스텔 기록 카드, 토스 스타일     | 기능 중심 UI               |

### 경쟁 분석

| 서비스      | 유형        | 강점                       | 약점                            |
| ----------- | ----------- | -------------------------- | ------------------------------- |
| 밀리의 서재 | 구독형      | 풍부한 콘텐츠, 대규모 유저 | 기록 기능 부족, 유료            |
| 트레바리    | 모임 플랫폼 | 체계적 모임 운영           | 비쌈 (월 5만원+), 오프라인 중심 |
| 북적북적    | 독서 기록앱 | 간편한 기록                | AI 없음, 모임 기능 없음         |
| StoryGraph  | 독서 트래킹 | 상세 통계, 추천            | 영어권, 한국 도서 부족          |

## 4. 수익화 전략 (Phase별)

### Phase 1: 무료 운영 (현재 ~ v2.0)

```
목표: 사용자 검증 + 커뮤니티 형성
수익: $0
비용: ~$5/월 (AI 비용만)
전략: 베타 초대 코드로 소규모 검증
```

### Phase 2: Freemium 모델 검토 (v2.1~)

**Free 플랜:**

- 월 20개 기록
- AI 대화 10회/월
- 모임 2개 참여
- 기본 통계

**Pro 플랜 ($2,900/월):**

- 무제한 기록
- AI 대화 무제한
- 모임 무제한
- 상세 독서 통계/인사이트
- 컬렉션 기능
- 우선 AI 응답

**전환 트리거:**

- 월 20개 기록 한도 도달 시 안내
- AI 대화 10회 소진 시 업그레이드 유도
- "Pro만 가능한 기능" 미리보기

### Phase 3: 추가 수익원 (v3.0~)

- **모임 주최자 도구** (프리미엄): 대규모 모임 관리, 출결, 통계
- **출판사 파트너십**: 신간 프로모션, 리뷰 이벤트
- **B2B 라이선스**: 기업 독서 동아리, 교육기관

## 5. 성장 전략

### 초기 (0~6개월): 시드 유저 확보

```
목표: MAU 50명
채널:
- 지인 초대 (초대 코드 기반 바이럴)
- 독서 모임 카카오 오픈채팅방 홍보
- 독서 관련 인스타그램/블로그 포스팅
핵심: "1명이 모임원 5명을 초대" 구조
```

### 중기 (6~12개월): 자연 성장

```
목표: MAU 500명
채널:
- SEO: 책 리뷰/감상 공개 피드 (Phase 3)
- 소셜 공유: 기록 카드 이미지 공유 기능
- 입소문: "AI가 기록 도와줘서 편해요" 바이럴
핵심: Single-Player 가치 먼저 (개인 기록), 소셜은 그 다음
```

### 장기 (1~3년): 플랫폼 확장

```
목표: MAU 5,000명
채널:
- 출판사/작가 파트너십 (신간 프로모션)
- 독서 챌린지/게이미피케이션
- PWA 앱스토어 등록
핵심: 콘텐츠 플라이휠 (기록 → 공유 → 유입 → 기록)
```
